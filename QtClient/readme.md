+ 日期: 2024/3/6

  - qt 实现国际化步骤
    - 1.在 cmake 文件中添加命令：qt_add_lupdate(QtClient TS_FILES ${TS_FILES})
    - 2.在项目构建，构建步骤中，勾选新出现的 lupdate 和 updata_translations
    - 3.选择构建，然后会有更新的 ts 文件出现
    - 4.选择在 qt 语言家打开，翻译，设置语言等操作
    - 5.查看代码中添加的翻译文件 qm 的资源目录，新建该目录
    - 6.选择文件发布，另存为在该目录
    - 7.在资源文件中加载该 qm 文件
    - 8.运行，翻译文件已经生效
